# BagelBoyz NJ

## Project Overview
Website and social media automation for **Bagel Boyz NJ** — a family-owned bagel shop in Hazlet, NJ with two locations. Founded in 2021 by Rob and Jess.

**Locations:**
- 694 Holmdel Rd, Hazlet, NJ 07730 — (732) 646-4455
- 1352 NJ-36 (Airport Plaza), Hazlet, NJ 07730 — (732) 335-1300
- Hours: 6 AM - 3 PM, 7 days/week

**Website:** bagelboyznj.com (hosted on Hostinger shared hosting)
**Instagram:** @bagelboyznj | **Facebook:** BagelBoyzNJ

## Tech Stack

### Website (Frontend)
- Static HTML/CSS/JS — no framework, no build step
- PHP for form handlers (catering inquiry, job applications)
- Google Fonts (Playfair Display, Inter, Dancing Script)
- Font Awesome 6.5 for icons
- Mobile-first responsive design
- Deployed to Hostinger shared hosting

### Social Media Automation (Backend)
- Node.js backend in `/backend/`
- PM2 for process scheduling (cron-based)
- AI content generation via OpenRouter or Anthropic API
- AI image generation: GPT-Image-1 → OpenRouter Flux → gradient fallback
- AI video generation: Kling 3.0 via Kie.ai API
- TTS: HeyGen Sammy → OpenAI TTS fallback
- FFmpeg for video assembly
- Meta Graph API v21.0 for Facebook/Instagram publishing
- Adapted from BoatLife.ai (`C:\Users\somar\Desktop\AI Claude Code Projects\BoatLife.ai`)

## NJ Bagel Terminology (Important)
- **Bagel Boyz** is spelled with a **Z** — always "Boyz" not "Boys"
- **Taylor Ham** (NOT pork roll) — Hazlet is in Monmouth County (North/Central NJ)
- **SPK** = Salt, Pepper, Ketchup — the standard NJ breakfast sandwich condiment
- **BEC** = Bacon Egg Cheese | **THEC** = Taylor Ham Egg Cheese
- **"On an everything"** = everything bagel
- **The boil** = boiling before baking, what makes a real NJ bagel

## Development

### Website
No build step — edit HTML/CSS/JS files directly. Preview by opening index.html in a browser.

Deploy to Hostinger: Upload files via FTP/SFTP or Hostinger file manager.

### Backend (Social Automation)
```bash
cd backend
cp .env.example .env        # Fill in API keys
npm install
npm run autopilot           # Manual test run
pm2 start ecosystem.config.js  # Start scheduled automation
```

### API Keys Needed (in backend/.env)
- `OPENROUTER_API_KEY` or `ANTHROPIC_API_KEY` — AI content generation
- `OPENAI_API_KEY` — Image generation (gpt-image-1)
- `KIE_API_KEY` — Video generation (Kling 3.0)
- `HEYGEN_API_KEY` — TTS voiceover
- `FB_PAGE_ID` + `FB_PAGE_ACCESS_TOKEN` — Facebook publishing
- `IG_ACCOUNT_ID` — Instagram publishing (uses FB page token)

## Conventions
- Use `var(--bb-*)` CSS custom properties for all brand colors
- Mobile-first: base styles for mobile, `@media (min-width: 768px)` for tablet+
- All pages share the same nav/footer structure
- Forms use `data-ajax` attribute for AJAX submission
- PHP form handlers return JSON `{ success: bool, message: string }`
- Social content must sound authentic NJ — no corporate language
